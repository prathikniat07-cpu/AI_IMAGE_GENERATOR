# AI_IMAGE_GENERATOR
 Built an AI system using Stable Diffusion + Qwen LLM It converts simple text prompts into high-quality images and also enables image editing using natural language (add objects, change styles, modify backgrounds).  Great hands-on experience in prompt engineering and combining LLMs with generative models.
